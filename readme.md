# SimplePS
A simple image processing toy build with tkinker and pillow
## 运行
  python3 ./Main.py

## 代码结构

```shell
├── AppFrame.py   	主界面
├── ImageCanvas.py	放置图像的组件
├── ImageHandler.py	图像处理类
├── Main.py		运行文件
├── TopMenu.py		顶部菜单栏
└── Utils.py		其他函数
```

## 依赖
sudo pacman -S tk
pip3 install pillow
pip3 install numpy
pip3 install matplotlib
